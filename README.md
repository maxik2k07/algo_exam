# ALGO ACADEMY — экзамен по алгоритмам и структурам данных

Одностраничный сайт-шпаргалка по 17 экзаменационным билетам: каждый билет
имеет краткое и полное объяснение (с псевдокодом), глоссарий и мини-квиз.

## Как залить на GitHub Pages
1. Создайте репозиторий на GitHub и загрузите содержимое этой папки
   (`index.html`, `e55fzb6s1lh51.jpg`, `README.md`).
2. В репозитории: **Settings → Pages → Branch: main / root → Save**.
3. Через минуту сайт будет доступен по адресу
   `https://<ваш-логин>.github.io/<репозиторий>/`.

Локально достаточно открыть `index.html` в браузере (двойной клик).
Шрифты подгружаются с Google Fonts, интернет нужен только для них.
